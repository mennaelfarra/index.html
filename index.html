🌍 Interactive Map Project

This interactive web map presents geographic information in a modern and accessible way. The project was developed using Python, Google Colab, and GIS tools, then published online with GitHub Pages.

GIS | Python | Leafmap | Web Mapping

📌 Project Overview

This project demonstrates how interactive mapping can be used to visualize spatial data clearly and effectively on the web.

🛠 Technologies Used

Python, Google Colab, HTML, GitHub Pages, and GIS mapping libraries were used to build and publish this project.

🎯 Purpose

The goal is to create an interactive map experience that is easy to explore, visually clean, and accessible to everyone online.
